# Nordic Haus - 北欧家具官网

一个现代化的斯堪的纳维亚风格家具展示网站，采用 Vue 3 + Vite + Tailwind CSS 技术栈构建，具有响应式设计和深色模式切换功能。

## 🌟 项目特色

- **现代技术栈**: Vue 3 + Vite + Tailwind CSS
- **响应式设计**: 完美适配桌面和移动设备
- **深色模式**: 支持本地存储的主题偏好设置
- **交互体验**: 流畅的交互动画和悬停效果
- **优雅设计**: 北欧简约风格的设计美学

## 📁 项目结构

```
nordic-haus/
├── public/                 # 静态资源
│   ├── favicon.svg         # 网站图标
│   └── icons.svg           # SVG 图标
├── src/
│   ├── assets/             # 资源文件
│   ├── components/         # Vue 组件
│   │   ├── Header.vue      # 导航头部组件
│   │   ├── Main.vue        # 主内容组件
│   │   └── Footer.vue      # 页脚组件
│   ├── App.vue             # 根组件
│   ├── main.js             # 应用入口
│   └── style.css           # 全局样式
├── index.html              # HTML 模板
├── package.json            # 项目配置
├── vite.config.js          # Vite 配置
└── pnpm-lock.yaml          # 依赖锁文件
```

## 🚀 快速开始

### 环境要求

- Node.js 18+
- pnpm (推荐) 或 npm

### 安装依赖

```bash
pnpm install
# 或
npm install
```

### 开发模式

```bash
pnpm dev
# 或
npm run dev
```

访问 [http://localhost:5173](http://localhost:5173) 查看网站

### 构建生产版本

```bash
pnpm build
# 或
npm run build
```

### 预览生产版本

```bash
pnpm preview
# 或
npm run preview
```

## 🛠️ 技术栈

### 核心框架
- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 现代化前端构建工具

### 样式与设计
- **Tailwind CSS v4** - 实用优先的 CSS 框架
- **响应式设计** - 移动端优先的布局策略
- **深色模式** - 自动适应系统主题偏好

### 开发工具
- **@vitejs/plugin-vue** - Vue SFC 支持
- **@tailwindcss/vite** - Tailwind CSS Vite 插件

## ✨ 功能特性

### 主题切换
- 支持亮色/深色主题切换
- 自动保存用户主题偏好到本地存储
- 遵循系统主题设置

### 导航体验
- 毛玻璃导航栏效果
- 响应式导航菜单
- 平滑的悬停动画

### 产品展示
- 家具系列分类展示
- 服务支持信息
- 设计师介绍
